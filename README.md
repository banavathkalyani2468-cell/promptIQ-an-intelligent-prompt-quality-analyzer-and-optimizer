# PromptIQ – An Intelligent Prompt Quality Analyzer and Optimizer

## 📌 Abstract

PromptIQ is a web-based application designed to improve the quality of user prompts before sending them to AI systems. Many users struggle to write clear and effective prompts, which leads to inaccurate or incomplete AI responses. PromptIQ analyzes the user input, identifies unclear or weak areas, and generates an optimized version of the prompt. The system helps users create better prompts and receive more meaningful AI-generated outputs through a simple and user-friendly interface.

# ⚙️ How PromptIQ Works

1. User enters a prompt in the input interface.
2. The frontend sends the prompt to the backend server.
3. The backend processes the prompt using OpenRouter API and LLaMA models.
4. The system analyzes grammar, clarity, and prompt quality.
5. An optimized version of the prompt is generated.
6. The improved prompt and AI response preview are displayed to the user.
7. Prompt history is stored in PostgreSQL database for future reference.

# 💻 How to Execute the Project

## Step 1: Clone the Repository

```bash
git clone <repository-link>
```

## Step 2: Open Project Folder

```bash
cd promptIQ-an-intelligent-prompt-quality-analyzer-and-optimizer
```

## Step 3: Install Dependencies

```bash
npm install
```

## Step 4: Configure Environment Variables

Create a `.env` file and add:

```env
DATABASE_URL=your_postgresql_url
OPENROUTER_API_KEY=your_api_key
```

## Step 5: Start the Backend Server

```bash
npm run dev
```

## Step 6: Open in Browser

```bash
http://localhost:3000
```
